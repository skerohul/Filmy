# Filmy – JavaScript projekt

Jednoduchý projekt v JavaScriptu pracující s daty o filmech.  
Aplikace je tvořena jednou HTML stránkou a jedním JS souborem, bez použití externích knihoven.

Data o filmech jsou uložena v poli objektů. Program ukazuje práci se strukturami `Object`, `Set` a `Map`, dále filtraci, převod dat, agregace (průměr, minimum, maximum) a vyhledávání.

Veškeré výstupy se vypisují do konzole webového prohlížeče.

# Spuštění
Otevřete soubor `filmy.html` v libovolném webovém prohlížeči a podívejte se do konzole.
